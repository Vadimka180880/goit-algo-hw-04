# Порівняльний аналіз алгоритмів сортування

## Опис
Цей проект містить порівняльний аналіз трьох алгоритмів сортування: сортування злиттям, сортування вставками та Timsort. Алгоритми тестувалися на наборах даних різного розміру для визначення їхньої ефективності за часом виконання.

## Алгоритми
- **Сортування злиттям** (Merge Sort): O(n log n)
- **Сортування вставками** (Insertion Sort): O(n^2) в гіршому випадку, O(n) в кращому випадку
- **Timsort**: O(n log n) в середньому та гіршому випадку, O(n) в кращому випадку

## Результати
Емпіричні результати підтверджують теоретичні оцінки складності алгоритмів. Timsort показав найкращі результати завдяки поєднанню сортування злиттям і вставками, що робить його ефективним для різних типів даних.

## Висновки
Timsort є найбільш ефективним алгоритмом серед розглянутих, що підтверджує правильність вибору цього алгоритму для реалізації сортування у Python. Сортування вставками виявилося найменш ефективним на великих масивах через квадратичну складність.

## Запуск проекту
Для запуску проекту необхідно мати встановлений Python та бібліотеки `matplotlib`. Запустіть скрипт `sorting_analysis.py` для виконання аналізу та побудови графіків.
